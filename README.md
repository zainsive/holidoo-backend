# Using Calendrific Api

Using calenndrific api to get holidays and get list of supported countries

## Tools

- express
- eslint
- morgan
- node-cache
- supertest
- ts-jest
- zod

## Run Locally

Clone the project

```bash
  git clone https://github.com/zainsive/holidoo-backend.git
```

Go to the project directory

```bash
  cd holidoo-backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run serve
```

Run test cases

```bash
  npm run test
```
